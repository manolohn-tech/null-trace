# Null-Trace 🕵️‍♂️

A professional steganography web suite that conceals encrypted data inside files using AES-256-GCM and LSB techniques.

## 🚀 Features
- 🔐 AES-256-GCM Encryption
- 🖼️ LSB Steganography
- 🧩 Single-page responsive design
- 🌐 Fully static and deployable

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript

## 📦 Deployment
Hosted using GitHub Pages.
